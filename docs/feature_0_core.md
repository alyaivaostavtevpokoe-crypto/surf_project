# Документ фичи: Инициализация ядра проекта и слоя данных (Core & Data Layer)

**Цель:** Создать базовую архитектурную структуру клиентского приложения на Flutter, реализовать Data-модели сущностей и запустить сервис имитации бэкенда (mocking).

## Что было реализовано:
1. Создана файловая структура папок проекта согласно Clean MVVM (`data/models`, `data/services`, `presentation`, `core`).
2. Описаны сущности `MasterModel`, `SlotModel`, `BookingModel` со встроенными методами сериализации (`fromJson`/`toJson`).
3. Разработан класс `MockApiService`, имитирующий сетевые запросы к API с искусственной задержкой в 800мс. Данные содержат тест-кейсы: штатные заезды, полностью заполненный заезд, а также заезд со статусом `isCancelledByWorkshop: true`.
